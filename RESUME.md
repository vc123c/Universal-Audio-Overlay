# Resume Notes

## Project Title

Universal Audio Overlay

## One-Line Description

Built a Windows desktop overlay for media control, source selection, album art, reactive visualization, and game-friendly click-through interaction.

## Resume Bullets

- Built an Electron-based Windows media overlay using Windows Runtime media APIs and Core Audio session control.
- Implemented global hotkeys, always-on-top transparent windows, source selection, playback controls, timeline seeking, and per-source volume control.
- Designed responsive compact layouts for tiny overlay windows with optimistic UI updates to reduce perceived input latency.
- Added album-art enrichment for music, podcasts, YouTube thumbnails, and audiobook metadata fallback.
- Integrated PowerShell helper processes with persistent IPC to reduce process-spawn overhead and improve responsiveness.

## Tech Stack

Electron, JavaScript, HTML/CSS, PowerShell, Windows Runtime APIs, Windows Core Audio APIs
