# Roadmap

## Near Term

- Add installer packaging with Electron Builder or Electron Forge.
- Add app icon and tray menu.
- Add a first-run help screen.
- Add crash-safe logging to a local file.
- Add a settings window for hotkeys and visual options.

## Later

- Native helper service for faster media/session polling.
- Optional signed audio backend for real EQ or effects.
- macOS prototype using native Now Playing APIs.
- Automated visual regression tests for compact layouts.
- GitHub Actions build pipeline for releases.
